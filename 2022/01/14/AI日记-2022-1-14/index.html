Breadth-First Search(广度优先搜索)

首先看一下最基础的BFS

import queue
frontier = Queue()
frontier.put(start)
reached = set()
reached.add(start)

while not frontier.empty():
    current = frontier.get()
    for next in graph.neighbors(current):
        if next not in reached:
            frontier.put(next)
            reached.add(next)

为了记录路径，我们把reached列表改为came_from字典

frontier = Queue()
frontier.put(start)
came_from = dict()
came_from[start] = None

while not frontier.empty():
   current = frontier.get()
   for next in graph.neighbors(current):
      if next not in came_from:
         frontier.put(next)
         came_from[next] = current

这是重构路径的算法

current = goal 
path = []
while current != start: 
   path.append(current)
   current = came_from[current]
path.append(start) # optional
path.reverse() # optional

上述算法有一个缺点，会将整个图全部搜索完，我们可以搜索到目标就停止

frontier = Queue()
frontier.put(start)
came_from = dict()
came_from[start] = None

while not frontier.empty():
   current = frontier.get()
   # -------------
   if current == goal: 
      break           
   # -------------
   for next in graph.neighbors(current):
      if next not in came_from:
         frontier.put(next)
         came_from[next] = current

Dijkstra’s Algorithm(迪杰斯特拉算法)

在BFS中，我们默认图中每条边的权值是相同的。当边的权值不同时，可以用Dijkstra算法解决

frontier = PriorityQueue()
frontier.put(start, 0)
came_from = dict()
cost_so_far = dict()
came_from[start] = None
cost_so_far[start] = 0

while not frontier.empty():
   current = frontier.get()

   if current == goal:
      break
   
   for next in graph.neighbors(current):
      new_cost = cost_so_far[current] + graph.cost(current, next)
      if next not in cost_so_far or new_cost < cost_so_far[next]:
         cost_so_far[next] = new_cost
         priority = new_cost
         frontier.put(next, priority)
         came_from[next] = current

Heuristic search(启发式搜索)

Dijkstra算法计算的是一个节点到其他所有节点的最短路径，而我们通常只需要知道一个点到另一个点的最短路径。所以我们可以定义一个启发函数(heuristic function)来描述当前节点和目标节点的距离。
比如用最简单的曼哈顿距离：

def heuristic(a, b):
   # Manhattan distance on a square grid
   return abs(a.x - b.x) + abs(a.y - b.y)

仅考虑启发函数就是贪心算法(Greedy Best First Search):

frontier = PriorityQueue()
frontier.put(start, 0)
came_from = dict()
came_from[start] = None

while not frontier.empty():
   current = frontier.get()

   if current == goal:
      break
   
   for next in graph.neighbors(current):
      if next not in came_from:
         priority = heuristic(goal, next)
         frontier.put(next, priority)
         came_from[next] = current

A* Algorithm

A* 算法结合Dijkstra算法和贪心算法，既考虑离目标的距离，也考虑离起始点的距离。

frontier = PriorityQueue()
frontier.put(start, 0)
came_from = dict()
cost_so_far = dict()
came_from[start] = None
cost_so_far[start] = 0

while not frontier.empty():
   current = frontier.get()

   if current == goal:
      break
   
   for next in graph.neighbors(current):
      new_cost = cost_so_far[current] + graph.cost(current, next)
      if next not in cost_so_far or new_cost < cost_so_far[next]:
         cost_so_far[next] = new_cost
         priority = new_cost + heuristic(goal, next)
         frontier.put(next, priority)
         came_from[next] = current

A* 算法是两全其美的。只要启发式算法没有高估距离，A* 就会找到一条最佳路径。

A* is optimal if

  • h(n) is admissible (never overestimates the
    true cost)
  • h(n) is consistent (for every node n and
    successor n’ with step cost c,h(n)≤h(n’)+c)
